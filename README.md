# Code Assistant - Интеллектуальный помощник для генерации и проверки кода

## Описание проекта
Система для автоматической генерации и проверки учебных заданий по программированию с использованием нейронных сетей на базе архитектуры трансформер.

## Основные функции
- Генерация кода по текстовому описанию
- Проверка синтаксиса и соответствия кода заданию
- Анализ стиля программирования
- Веб-интерфейс для преподавателей и студентов

## Технологии
- Python 3.8+
- PyTorch
- Django REST Framework
- Transformers (Hugging Face)
- HTML5/CSS3/JavaScript

## Установка и запуск
1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите сервер: `python main.py`
3. Откройте в браузере: `http://localhost:8000`

## Структура проекта
- `main.py` - основной файл приложения
- `src/api/` - API endpoints
- `src/models/` - модели машинного обучения
- `src/frontend/` - веб-интерфейс
